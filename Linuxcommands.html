ls

ls: the ls command tells the user what files are in the directory they are in. An example of this command would be ls -a which shows hidden files.

cd

cd: the cd command is used to go to a directory. An example would be if you wanted to go to the downloads folder from the home folder, you could type "cd Downloads".

mkdir

mkdir: the mkdir command is used when you need to create a directory. For example, if you wanted to create a directory called "ABC", you can type "mkdir ABC".

rmdir

rmdir: the rmdir is used to delete a directory. It can only be used to delete an empty directory. If it contains files, you cannot use rmdir. For example, if "ABC" was empty, you could use "rmdir ABC".

rm

rm: the rm command is used to delete files. For example, if you had a file named "rpistatus", you could do "rm rpistatus" to delete it.

touch

touch: the touch command is used to create a new file. It can be used for anything, such as empty txt files and zip files. For example, you could do "touch ABC.txt"

man

man: the man command is used to learn more about a command. It shows the manual pages of the command. For example, "man ls" shows the manual pages of the ls command.

cp

cp: the cp command is used to copy files through the command line. There are 2 arguments: the location of the file to be copied, and the where to copy.

mv

mv: the mv command is used to move files through the command line. The setup is similar to the cp command. For example, if you want to move lol.txt to downloads, you could do "mv lol.txt ~/Downloads"

cat

cat: the cat command is used to display the contents of the file. For example, if one of my texts was "hi my name is bob" and was named abc.txt, I could do "cat abc.txt" to show the contents of what I wrote."

sudo

sudo: the sudo command is to execute any command with administrative or root privileges. For example, you could do "sudo passwd" to make a new root password.
